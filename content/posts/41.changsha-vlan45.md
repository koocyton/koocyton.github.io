从弱电箱拉一根线到路由器，从路由器 lan3 连线给 iptv 机顶盒

每次启动执行下面脚本

lan3  和 vlanid 45 看你的环境，长沙电信应该是 45

```
#!/bin/sh

# 1. 创建 IPTV VLAN
ip link add link wan name wan.45 type vlan id 45
ip link set wan.45 up

# 2. 创建 bridge
ip link add name br-iptv type bridge
ip link set br-iptv up

# 3. 加入 VLAN
ip link set wan.45 master br-iptv

# 4. 从 LAN 剥离 lan3
ip link set lan3 nomaster

# 5. 加入 IPTV
ip link set lan3 master br-iptv

# 6. 开启组播
echo 1 > /sys/class/net/br-iptv/bridge/multicast_snooping
echo 2 > /sys/class/net/br-iptv/bridge/multicast_querier
```
