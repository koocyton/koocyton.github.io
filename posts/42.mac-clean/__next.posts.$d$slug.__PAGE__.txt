<h2>特性</h2>
<ul>
<li><strong>拖放分析</strong>：将应用程序包拖到窗口中的拖放区即可解析 <code>Info.plist</code>，读取显示名称与 <code>CFBundleIdentifier</code>。</li>
<li><strong>关联项扫描</strong>（在用户目录与部分系统路径中查找常见位置，避免全盘深度遍历）：
<ul>
<li>应用程序包本体</li>
<li>偏好设置（<code>Preferences</code> 下对应 plist）</li>
<li>保存的应用状态、Caches、Application Support、Logs、WebKit 等目录下以 Bundle ID 命名的条目</li>
<li>沙盒容器（<code>Containers</code>）</li>
<li>崩溃报告、诊断报告（<code>.ips</code>）</li>
<li>在 Application Support / Caches / Logs 等目录下按名称与 Bundle ID 片段做的<strong>浅层名称匹配</strong></li>
<li>部分应用可能写入的 <code>/var/log</code> 下匹配项</li>
</ul>
</li>
<li><strong>选择性卸载</strong>：每项可单独勾选；支持「全部保留」「全部卸载」快捷操作。</li>
<li><strong>安全删除</strong>：删除操作使用系统废纸篓 API，不是 <code>rm</code>，误删可在废纸篓中还原。</li>
<li><strong>图形界面</strong>：基于 egui/eframe，带中文界面与现代化深色主题；可打包为 <code>.app</code>，从访达双击启动（不依赖终端窗口）。</li>
</ul>
<p><strong>说明</strong>：扫描结果属于<strong>启发式</strong>列表，无法保证覆盖某应用的全部残留；卸载前请自行确认路径。若应用数据在其他自定义位置，不会自动出现。</p>
<hr>
<h2>系统要求</h2>
<ul>
<li><strong>macOS</strong>（建议 11.0 及以上，与打包脚本中的最低版本一致）</li>
<li>若从源码构建：已安装 <strong>Rust</strong>（<a href="https://rustup.rs/">rustup</a>）与 <strong>Xcode 命令行工具</strong>（含 <code>iconutil</code>、<code>sips</code>，用于生成 <code>.icns</code>）</li>
</ul>
<hr>
<h2>安装</h2>
<h3>方式一：自行从源码构建（命令行二进制）</h3>
<pre><code class="hljs language-bash"><span class="hljs-built_in">cd</span> macclean   <span class="hljs-comment"># 本仓库根目录</span>
cargo build --release
</code></pre>
<p>可执行文件：<code>target/release/macclean</code>。适合开发调试；若用 <code>cargo run</code> 启动，进程会挂在当前终端上。</p>
<h3>方式二：打包为 macOS 应用程序（推荐日常使用）</h3>
<p>在项目根目录执行：</p>
<pre><code class="hljs language-bash">bash packaging/macos/bundle.sh
</code></pre>
<p>脚本会：</p>
<ol>
<li><code>cargo build --release</code></li>
<li>根据 <code>assets/app_icon.png</code> 生成 <code>AppIcon.icns</code></li>
<li>在项目根目录生成 <strong><code>MacClean.app</code></strong></li>
</ol>
<p>将 <strong><code>MacClean.app</code></strong> 拖入「应用程序」文件夹即可。从<strong>启动台</strong>或<strong>访达</strong>双击打开，一般不会弹出终端窗口。</p>
<blockquote>
<p>若系统提示来自未认证开发者：可在访达中对 <code>MacClean.app</code> <strong>右键 → 打开</strong>，或在「系统设置 → 隐私与安全性」中允许运行。自行签名或公证可进一步减少提示（需 Apple 开发者账号）。</p>
</blockquote>
<hr>
<h2>使用说明</h2>
<ol>
<li>启动 <strong>MacClean</strong>（<code>.app</code> 或命令行二进制均可）。</li>
<li>将要分析的应用 <strong><code>.app</code> 拖入窗口中的拖放区</strong>（或拖到窗口上，行为以当前版本为准）。</li>
<li>等待列表加载：上方会显示应用显示名与 Bundle ID；下方表格列出扫描到的路径、类别与估算大小。</li>
<li><strong>勾选</strong>需要移入废纸篓的项目（默认多为全选，可按需取消）。</li>
<li>点击 <strong>「卸载选中项」</strong>，等待进度完成。</li>
<li>需要时到 <strong>废纸篓</strong> 中检查或恢复文件。</li>
</ol>
<p><strong>提示</strong></p>
<ul>
<li>若某次扫描结果为空或偏少，可能是该应用未在常见路径留下可识别文件，或名称与 Bundle ID 与启发式规则不匹配。</li>
<li>卸载系统组件或共享库相关路径前请格外谨慎。</li>
</ul>
