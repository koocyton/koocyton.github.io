3:T1a8b,<h1>Rust Download Manager</h1>
<h2>下载地址</h2>
<p><img src="https://github.com/koocyton/rust-bt-downloader" alt="https://github.com/koocyton/rust-bt-downloader"></p>
<p>A native GUI download manager for <strong>HTTP/HTTPS</strong> (multi-connection) and <strong>BitTorrent</strong> (magnet / <code>.torrent</code>), built with Rust, <strong>iced</strong>, and <strong>librqbit</strong>. Inspired by classic download managers (e.g. IDM-style workflow).</p>
<hr>
<h2>中文说明</h2>
<h3>功能概览</h3>
<ul>
<li><strong>HTTP/HTTPS</strong>：多连接分块下载、断点续传（分块临时目录）、HEAD/Range 探测、单连接回退。</li>
<li><strong>BitTorrent</strong>：磁力链接、HTTP(S) 种子、<strong>本地 <code>.torrent</code> 文件</strong>（含系统文件对话框选择）。</li>
<li><strong>任务管理</strong>：开始、暂停、停止、移除任务、删除任务及本地文件；任务状态持久化（<code>tasks.json</code>）。</li>
<li><strong>界面</strong>：深色主题、任务列表、详情面板（HTTP 显示连接与速度；BT 显示 Peers、分块、ETA、上传等）、状态栏与全局 DHT 摘要。</li>
<li><strong>字体</strong>：可放置 <code>assets/NotoSansSC-Regular.ttf</code> 或使用系统字体路径（见 <code>src/font.rs</code>）。</li>
</ul>
<h3>不支持</h3>
<ul>
<li><strong>FTP</strong>（<code>ftp://</code>）：当前仅使用 HTTP 客户端，不支持 FTP 直链；请用支持 FTP 的工具或浏览器扩展。</li>
<li><strong>速度限制</strong>（<code>speed_limit</code>）：配置项存在，引擎侧尚未完全接入。</li>
</ul>
<h3>运行</h3>
<pre><code class="hljs language-bash"><span class="hljs-built_in">cd</span> rust-download
cargo run --release
</code></pre>
<p>首次编译会拉取较多依赖，请耐心等待。请在<strong>项目根目录</strong>运行，以便加载 <code>assets/</code> 下的字体（可选）。</p>
<h3>配置</h3>
<p>配置文件位于 <code>~/.config/rust-download/config.json</code>（Windows 为 <code>%APPDATA%\rust-download\config.json</code>）。首次运行会自动创建。</p>

</tr>

<td>每个 HTTP 任务最大连接数</td>

<h2>English</h2>
<h3>Overview</h3>
<ul>
<li><strong>HTTP/HTTPS</strong>: Multi-connection chunked downloads, resume via temp chunk files, HEAD/Range probing, single-stream fallback when needed.</li>
<li><strong>BitTorrent</strong>: Magnet links, <code>.torrent</code> over HTTP(S), <strong>local <code>.torrent</code> files</strong> (including native file picker via <code>rfd</code>).</li>
<li><strong>Tasks</strong>: Start, pause, stop, remove task, delete task + files; persistent task state (<code>tasks.json</code>).</li>
<li><strong>UI</strong>: Dark theme, task list, detail panel (HTTP: connections &#x26; speed; BT: peers, pieces, ETA, upload, etc.), status bar with global DHT summary.</li>
<li><strong>Fonts</strong>: Optional <code>assets/NotoSansSC-Regular.ttf</code> or system font paths (see <code>src/font.rs</code>).</li>
</ul>
<h3>Not supported</h3>
<ul>
<li><strong>FTP</strong> (<code>ftp://</code>): Only HTTP is implemented for direct URLs; use an FTP-capable client.</li>
<li><strong>Speed limit</strong> (<code>speed_limit</code>): Field exists in config; not fully wired in the engine yet.</li>
</ul>
<h3>Run</h3>
<pre><code class="hljs language-bash"><span class="hljs-built_in">cd</span> rust-download
cargo run --release
</code></pre>
<p>The first build downloads many crates. Run from the <strong>project root</strong> if you want bundled fonts under <code>assets/</code>.</p>
