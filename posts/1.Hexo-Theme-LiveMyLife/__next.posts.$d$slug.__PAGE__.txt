<h2>快速开始</h2>
<p>为了方便起见，我发布了整个项目，因此您可以按照下面的说明进行操作，然后就可以轻松自定义您自己的博客！</p>
<p>让我们开始!!!</p>
<h3>安装 Node.js 和 Git</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_">#</span><span class="bash">For Mac</span>
brew install node
brew install git
</code></pre>
<blockquote>
<p>Windows：下载 &#x26; 安装 Node.js。-> <a href="https://nodejs.org/zh-cn/download/">Node.js</a></p>
<p>Windows：下载 &#x26; 安装 Git。-> <a href="https://git-scm.com/download/win">Git</a></p>
</blockquote>
<h3>安装 Hexo</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_">$ </span><span class="bash">npm install -g hexo-cli</span>
</code></pre>
<blockquote>
<p>什么是 <a href="https://hexo.io/docs/">Hexo</a>?</p>
<p>Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。</p>
</blockquote>
<h3>设置你的博客</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_">$ </span><span class="bash">hexo init blog</span>
</code></pre>
<blockquote>
<p>更多的命令 -> <a href="https://hexo.io/docs/commands">Hexo Commands</a></p>
</blockquote>
<h2>主题使用</h2>
<h3>初始化</h3>
<pre><code class="hljs language-shell">cd blog
rm -rf scaffolds source themes _config.landscape.yml _config.yml package.json yarn.lock #just keep node_modules
git clone https://github.com/V-Vincen/hexo-theme-livemylife.git
mv hexo-theme-livemylife/* ./
rm -rf hexo-theme-livemylife
npm install
</code></pre>
<h3>设置主题</h3>
<p>修改 <code>theme</code> 属性值，在 <code>_config.yml</code> 配置文件中。</p>
<pre><code class="hljs language-yml"><span class="hljs-comment"># Extensions</span>
<span class="hljs-comment">## Themes: https://hexo.io/themes/</span>
<span class="hljs-comment">## Plugins: https://hexo.io/plugins/</span>
<span class="hljs-attr">theme:</span> <span class="hljs-string">livemylife</span>
</code></pre>
<h3>启动服务</h3>
<pre><code class="hljs language-shell">hexo generate # or hexo g
hexo server   # or hexo s
</code></pre>
<p>启动本地服务器。 默认访问地址 <code>http://localhost:4000/</code>。</p>
<blockquote>
<p>更多命令 -> <a href="https://hexo.io/docs/commands">Hexo Commands</a></p>
</blockquote>
