13:T12fe,<h5>前言</h5>
<p>FeignClient 在调用服务时，目标端发生错误。
返回:</p>
<pre><code class="hljs language-json"><span class="hljs-punctuation">{</span>
  <span class="hljs-attr">"code"</span><span class="hljs-punctuation">:</span><span class="hljs-string">"100005"</span><span class="hljs-punctuation">,</span>
  <span class="hljs-attr">"message"</span><span class="hljs-punctuation">:</span><span class="hljs-string">"error message"</span><span class="hljs-punctuation">,</span>
  <span class="hljs-attr">"data"</span><span class="hljs-punctuation">:</span><span class="hljs-string">""</span>
<span class="hljs-punctuation">}</span>

FeignClient 接口定义为<span class="hljs-punctuation">:</span>

```java
@FeignClient(name = <span class="hljs-string">"feignLogin"</span><span class="hljs-string">")
public interface LoginFeign {

    @GetMapping(value = "</span>/api/feignUser<span class="hljs-string">")
    Result&#x3C;User> feignUser();
}
</span></code></pre>
<p>这是 Feign 在序列化 <code>Result&#x3C;User></code> 时，会发生错误，
因为 "" 反序列化 User ，jackson 抛出异常。</p>
<p>经过反复测试，可以在拦截器上做统一的错误处理。</p>
<h5>解决</h5>
<ul>
<li>FeignClient config 配置</li>
</ul>
